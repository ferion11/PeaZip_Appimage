# PeaZip_Appimage

PeaZip Appimage for linux 64bits

#### 1- Download "PeaZip-VERSION-x86_64.AppImage"
#### 2- Make executable:
- `$ chmod +x PeaZip-VERSION-x86_64.AppImage`
#### 3- Run it:
- `$ ./PeaZip-VERSION-x86_64.AppImage`
